# **App Name**: Al Fajr

## Core Features:

- Super Admin Login: Restrict login access to a single super admin account.
- Add Member: API endpoint to add new members with the specified details (fullName, mobileNumber, address, membershipType, dateOfJoining, status, openingBalance, adharNumber, photo).

## Style Guidelines:

- Primary color: Deep Blue (#3F51B5) to evoke a sense of trust and professionalism.
- Background color: Light Gray (#F5F5F5) for a clean and modern look.
- Accent color: Teal (#009688) for interactive elements and important calls to action.
- Body font: 'Inter' (sans-serif) for a modern, readable style appropriate for the text and forms.
